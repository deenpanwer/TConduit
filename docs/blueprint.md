# **App Name**: RetroSheet

## Core Features:

- Input Field: A central text input field on the page for users to type in their data.
- Data Submission: Submits the typed content to the specified sheetdb.io endpoint upon pressing 'Enter'.
- API Connection: Connect to the sheetdb.io API endpoint (https://sheetdb.io/api/v1/q1xovvwyyhvv0) to post user input.

## Style Guidelines:

- Primary color: Blue (#4285F4) to capture the essence of classic Google.
- Background color: Very light gray (#F5F5F5) for a clean, simple background.
- Accent color: Green (#34A853) for subtle highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font, is used to maintain simplicity and readability. This choice is appropriate as the app will be used for both headlines and body text.
- Center the input field on the page to emulate the classic Google search interface.
- Subtle fade-in animation when the page loads for a smooth user experience.
